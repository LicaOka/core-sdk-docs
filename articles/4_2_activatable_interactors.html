    Interactors

    Interactors are simply regions in application's window with some associated action or actions which can be triggered when user is looking at them.

    To get started, let's create an interactor which could tell whether user is looking at applications's window or not. First thing we need is the instance of the Host class. 

    using Tobii.EyeX;
...

var host = new Host();

    To get knowledge about application's window you need a handle to it. There could be various ways to obtain handle to a window, depending on which kind of application you are developing. For example you can use Process class from System.Diagnostics:

    ...
using System.Diagnostics;
...

...
var mainWindowHandle = Process.GetCurrentProcess().MainWindowHandle;
var interactorAgent = host.InitializeUnboundAgent(mainWindowHandle);

    As you noticed we have created an instance of UnboundInteractorAgent here. We will tell you more about interactor agents later, but for now you can think of them as of helper classes which provides you with simple and elegant way of defining interactors for specific window. We have supplied here our application's main window handle which means interactorAgent wil help us create interactors for that window by default.

    At last, we will define our interactor. And because interactor is region it has to have bounds. Bounds should be defined as intance of Tobii.EyeX.Client.Rect struct. Let's assume that you know bounds of your application's main window: 

    ...
using Tobii.EyeX.Client;
...

...
var interactorBounds = new Rect {
    X = mainWindowBounds.X,
    Y = mainWindowBounds.Y,
    Width = mainWindowBounds.Width,
    Height = mainWindowBounds.Height
};

var interactor = interactorAgent.AddInteractorFor(interactorBounds);

    Last thing is to define actual behavior. Let's assume that whenever user looks or stop looking at the application's main window you going to log that information to the command line:

    using System;
...

...
interactor
    .WithGazeAware()
    .HasGaze(() => Console.WriteLine("User has started looking at the window at {0}", DateTime.Now))
    .LostGaze(() => Console.WriteLine("User has stopped looking at the window at {0}", DateTime.Now));
