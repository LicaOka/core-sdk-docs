Getting started

This first chapter will teach you how to start using eye tracking in a brand new application or in your existing one.

Installation

No development for eye tracking could be successfully done without device itself. If you have a laptop or a monitor with built-in eye tracker you are most probably ready to proceed with development. If you have peripheral eye tracker, then follow the instructions included with it to download and install the correct Tobii Engine runtime software. Make sure the eye tracker and software is working correctly.

Creating console application

If you want to begin from scratch, the easiest way will be to create a new console application using C#. Open your Visual Studio desktop app and make sure you created one.

Adding eye tracking to an existing application

You don't necessarily need to create a new application to start using eye tracking. Maybe you already have your amazing application and only lack some gaze based features to it. Depending on what it is you can jump into Getting started with WPF or Getting started with WinForms. If it is neither of those you are welcome to continue to read this chapter.

Installing Unified Foundation

We recommend to reference Unified Foundation from NuGet server. To do so, right click on you project's reference folder and click Manage NuGet Packages..., search for Unified Foundation and install it. After doing so you will notice that project.json has been added (or updated) in the project folder, and your project is now referencing Tobii.EyeX.Model.dll and Tobii.EyeX.Net.dll. We could also recommend to configure your build to use x86 architecture instead of ANY CPU to prevent potential warnings.

Let's code

Now that you have installed all components we will help you to write your first eye tracking feature. It is often that people who have heard about eye tracking want to find out where they looking the first time they are using the feature. So we going to do exactly that - output coordinates of your gaze on screen.

Add this to your Program.cs:

using Tobii.EyeX;

// ...

public static void Main(string[] args)
{
    var host = new Host();
    var gazePointDataStream = host.Streams.CreateGazePointDataStream();
    gazePointDataStream.GazePoint((gazePointX, gazePointY, _) => Console.WriteLine("X: {0} Y:{1}", gazePointX, gazePointY));
}

// ..

When you run this code, if everything were installed correctly, you will be able to see the stream of coordinates of your gaze. Also because we have assumed that you could add eye tracking to your existing application make sure that if you do not have Console you output this information in suitable for you way. Let's see what we actually did just now.

First we have instantiated the Host class which serves as entry point into Unified Foundation. It provides you with different kind of features, but also it creates connection between your application and the Interaction Engine.

Next we have used Host to instantiate stream of gaze data, which contains coordinates of your gaze location and timestamp.

Finally we have subscribed to the stream with call-back function, which will be invoked when new data will be pushed to the stream.
