Class UnboundInteractorAgent

Default implementation of an unbound IInteractorAgent using UnboundInteractor as interactor type. Everything is taken care of. UnboundInteractors are moved by using the Move, MoveTo and MoveCenterTo extension methods or by explicitly setting the CachedBounds property.
